Projects

Appetite Crush

Appetite Crush is a MEAN stack web application that provides personalized recipes based on user preferences. Built with Angular for the front-end and Node.js, Express, and MongoDB for the back-end, it includes user authentication and dynamic recipe search functionality. The app enhances the user experience by offering an interactive and intuitive interface to explore a variety of recipes based on different criteria.

The project helps users discover recipes suited to their tastes and dietary restrictions, making it a useful tool for anyone interested in cooking.

Turing Hut

Turing Hut is a website developed for a college programming club, built using React JS and Gatsby. It serves as a platform for the club, featuring pages for events, articles, team details, and more. The website provides a comprehensive resource for club activities and includes a blog system for members to share coding tips and project showcases, encouraging collaboration among students.

The website enhances engagement in the club by offering up-to-date event information and interactive content, helping foster a stronger community.

Movie Review Classification using KNN

I built a K-Nearest Neighbors (KNN) algorithm from scratch to classify movie reviews as positive or negative. The project involved data pre-processing, feature extraction, and fine-tuning the model's hyperparameters to improve accuracy. The project offers valuable insights into sentiment analysis and is an example of working with unsupervised learning.

The project helped me understand how KNN can be applied to text data, offering an excellent foundation for exploring natural language processing techniques.

Handwritten Digit Recognition with K-Means Clustering

In this project, I applied K-Means clustering to recognize handwritten digits. The focus was on unsupervised learning techniques for image and pattern recognition. I visualized the results with Python libraries like Matplotlib to demonstrate the clustering outcomes, which helped in understanding the decision-making process of the algorithm.

This project provided insights into the power of clustering algorithms and their application in image recognition tasks.
